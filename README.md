# 🎉 Fun Calculator 🎉

Welcome to the **Fun Calculator** – a simple and playful program to perform basic arithmetic operations like addition, subtraction, multiplication, and division on two numbers! 😎

---

## Features

- Takes two numbers as input (supports decimals!).
- Calculates their:
  - Sum ➕
  - Difference ➖
  - Product ✖️
  - Quotient ➗
- Displays the results in a fun and easy-to-understand way.

---

## How to Use

1. Run the script.
2. Enter the first number (can be a decimal).
3. Enter the second number (can be a decimal).
4. See the results printed out instantly.

**Example:**

```
Enter the first number: 10
Enter the second number: 4
Results of your two numbers:
Sum: 14.0
Difference: 6.0
Product: 40.0
Quotient: 2.5
```

---

## Notes

- The calculator assumes the second number is **not zero** to avoid division errors.
- You can modify the code to handle exceptions if needed.

---

## Code Overview

- Uses `float()` to accept numbers with decimal points.
- Performs all four basic arithmetic operations.
- Prints results with clear labels and emojis for fun!
